# Curated Fashion Platform

A high-end, fashion-forward minimalist experience.

## Project Structure

- `client`: Frontend built with React and Vite.
- `server`: Backend API and services.

## Setup

1. Install dependencies:
   ```bash
   # In client/
   npm install
   # In server/
   npm install
   ```
2. Run development servers:
   ```bash
   # In client/
   npm run dev
   # In server/
   npm run dev (if available) or npm start
   ```
